# Benthic Image Preprocessor

## Run
```shell
uv run preprocessor ui
```

## Tests
```shell
uv run pytest
```

## Tests & Coverage
```shell
uv run pytest --cov
```

## Determine version
```shell
uv run hatch version
```

## Lint
```shell
uv run ruff check
```

## Format
```shell
uv run ruff format
```
